# Project Instructions

## Ignore directories

Do not read, analyze, modify, or search inside these directories:

- `vendor/`
- `node_modules/`

These contain third-party dependencies and should be treated as external code.

When exploring the codebase, focus only on:

- `inc/`
- `blocks/`
- `templates/`
- `parts/`
- `patterns/`
- `functions.php`
- `theme.json`
- `tests/`
- configuration files (`composer.json`, `package.json`, `vite.config.js`, etc.)

## Project Context

This is a WordPress Full Site Editing (FSE) theme.

Technology stack:
- PHP 8+
- WordPress
- Gutenberg blocks
- React
- @wordpress/scripts
- Vite
- Sass
- PHPUnit + Brain Monkey
